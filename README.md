# Data Gathering Practice

## Overview
This repository contains a data gathering notebook where different datasets are loaded and explored using Python and Pandas.  
The purpose of this notebook is to practice collecting data, understanding its structure, and preparing it for further analysis.

This work is created for academic practice and learning.

## What is done in this notebook
- Loading data from CSV files
- Inspecting rows and columns
- Checking data types and missing values
- Basic data exploration
- Simple groupby operations
- Exporting processed data to CSV

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Purpose
The main goal is to build understanding of the data gathering step in the data analysis workflow.  
This includes reading data, exploring it, and preparing it for future analysis.

## Usage
Open the notebook and run the cells to see how data is loaded and processed.
